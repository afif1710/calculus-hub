# CalcHub (Calculator Hub)

A fast, dark-themed collection of practical calculators grouped by category, with expandable category cards and favorites.

## Features

- Category grid with expand/collapse calculator lists
- Favorite/star calculators for quick access
- Responsive UI with consistent spacing and typography
- Deterministic calculators (no external APIs required)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started (Local)

### Prerequisites

- Node.js (LTS recommended)
- npm

### Install & Run

```bash
git clone https://github.com/afif1710/calculator-hub.git
cd calculator-hub
npm install
npm run dev

```
